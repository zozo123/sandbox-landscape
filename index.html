Use cases

Pick the workload first.

Use the shortest path that matches the task.

Use case Need Good fit
Code interpreter Short, isolated execution E2B / Modal / OpenAI / Anthropic / Gemini
Evals and RL Parallel runs and harness control Harbor / Modal / Daytona / Tensorlake
CI and PR checks Cheap, repeatable validation GitHub Actions / ARC / Docker Sandboxes
Long-running agents Persistent state and recovery Daytona / Runloop / Islo / exe.dev
Brokered enterprise workflows Policy, credentials, routing Crabbox / custom broker / ARC
